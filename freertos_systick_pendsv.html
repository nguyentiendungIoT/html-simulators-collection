// ISR Handler cho SysTick Timer
void xPortSysTickHandler( void )
{
    /* 1. Vào Critical Section (ngắt mềm) */
    portDISABLE_INTERRUPTS();
    
    /* 2. Báo Kernel tăng Tick count */
    if( xTaskIncrementTick() != pdFALSE )
    {
        /* 3. Nếu cần switch, kích hoạt PendSV */
        portNVIC_INT_CTRL_REG = portNVIC_PENDSVSET_BIT;
    }
    
    /* 4. Thoát Critical Section */
    portENABLE_INTERRUPTS();
}

lightbulb Tại sao phải dùng PendSV?

Nếu thực hiện Context Switch ngay trong SysTick, quá trình này (lưu/phục hồi thanh ghi) sẽ diễn ra ở độ ưu tiên của SysTick.

Hệ quả xấu:

  • Các ngắt quan trọng khác (như DMA, UART) sẽ bị chặn cho đến khi Switch xong.
  • Gây ra hiện tượng "Interrupt Jitter" (trễ ngắt không đoán trước).
